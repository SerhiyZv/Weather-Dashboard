# Weather Dashboard

## Project Overview

Weather Dashboard application v1.0. The Weather Dashboard application is the single-window web-application to display current weather and 5-days forecast for cities search by user.
Application is using [OpenWeather API](https://openweathermap.org/api) po pull weather and forecast information for cities around the world.
HTML data is dynamically changed by JavaScript with data requested through fetch methods and parsed by JSON. Bootstrap was used to change UI with minimal CSS.
Also Moment.js is called to display current date and future dates for forecast.
Local storage was used to save data from previous user requests, and these requests displayed on previous search section.

## Technologies Used

* JavaScript
* HTML
* CSS

## APIs/Libraries Used

* [OpenWeather API](https://openweathermap.org/api)
* [Bootstrap](https://getbootstrap.com/)
